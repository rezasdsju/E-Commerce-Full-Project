# Set Up the Project

## Commands in Terminal for  Django Setup
- python -m venv venv
- pip install django djangorestframework psycopg2-binary python-dotenv
- djnago-admin startproject backend .
- python manage.py startapp store


## Commands for React SetUp
- Google Search: React installation via Vite
- npm create vite@latest frontend
- Select 'React'
- Select 'JavaScript'
- Select 'No' for 'Use rolldown vite(Experimental)'
- Select 'Yes' for 'Install with npm and start now'
